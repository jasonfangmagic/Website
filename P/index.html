Models to Analyze

I do know more models, but these models are some basic and useful models that I use very often.

Linear Regression model is not as accurate as machine learning models, but it can generate some meaningful managerial outcomes for decision-making.

In terms of machine learning models, I like decision tree and random forest. I use SVM, KNN, Naive Bayes and other models as well.

Deep learning models are my favorites. I don't have to collect a lot of data. I could just use data augmentation or just use some pre-train models. It's very useful in some other fields. I had used CNN model and OpenCV to create a self-driving simulation.

PCA, EFA, and K-Means are used for clustering. I use time series to predict future revenue.
