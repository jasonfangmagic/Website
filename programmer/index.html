Self-Driving Car

Since I’m very interested in the self-driving car industry, I used Python to create a Convolution Neural Networks (CNN) model to simulate a self-driving car. It’s very similar to the ongoing self-driving experiments conducted by Tesla, Google, Uber, and so on. My process is simpler, but the idea is the same.

1. Used a self-driving simulator provided by Udacity to generate the data.

2. Balanced the data and split the data into the training and validation data set.

3. Preprocessed the data and used OpenCV.

4. Used Convolution neural network CNN to create a model.

5. Tested the model by using the code provided by Udacity.

Software used: Python, Pycharm, Git, Udacity self-driving simulator

Packages used: tensorflow, keras, sklearn, cv2, pandas, numpy, os, matplotlib, imgaug

IMDB Rating Analysis
I used some linear regression and machine learning models to analyze an IMDB dataset which contains 14166 movies and 72 variables. These variables are audience ratings and demographic information.

I first created a linear regression model to find out which variables are crucial to the movie box office.

Then I used the Forward Stepwise Regression model and Full Model Search model to delete some variables to decrease over-fitting.

Finally, I used the decision tree and random forest model to find out which age group has the greatest influence on the movie rating by using the Variable Important Plot.

Software used: R Studio

Packages used: Linear Regression, Forward Stepwise Regression, Full Model Search, Decision Tree, Random Forest

Natural Language Processing is playing an important role in the machine learning field. I’d like to share a powerful NLP tool which is provided by IBM, IBM Watson. Watson is a question-answering computer system capable of answering questions posed in natural language, developed in IBM's DeepQA project by a research team led by principal investigator David Ferrucci.


It can easily be used in Python. You can just install ibm_watson and import the module that you want to use. There are 4 modules that I use very often, Text to Speech, Speech to Text, Translation, and Tone Analysis. Text to Speech is very accurate because the computer can read every single word. Tone Analysis could help you analyze your sentence’s tone. It’s been used in interpreting reviews and Twitter content. This powerful tool can help you find out whether the comment is positive or negative.
