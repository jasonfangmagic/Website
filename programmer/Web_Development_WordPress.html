OmniFun is a kid’s playground located in Maryland, Gaithersburg. Because other managers don’t know HTML, I chose to use WordPress to build this website. This website is designed to show parents the pictures, pricing, and policies. I also implemented Google Analytics which can track online viewers' information.

The interface of WordPress is very straightforward. You can just edit it on the left bar and you can see the layout on the right side.



I created the "Gallery" page for customers to view the pictures of the playground. Since I launched this web page, the visitor numbers have increased 10% in 3 months.



In order to let customers know more about the playground, I used Google Street View to take 30 panoramic pictures. People can view the playground before they come. This web page is very popular as well.





Software used: WordPress, Google Analytics
